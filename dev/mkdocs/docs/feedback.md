# Problemas conhecidos / feedback

Isso é esperado numa versão de teste — não desanime. Se algo der errado,
anote (ou tire print de):

- O que você estava tentando fazer.
- O que apareceu na tela (mensagem de erro, comportamento inesperado).
- Se possível, o arquivo de relatório que o gerador produz
  (`RELATORIO.html` ou `RELATORIO_REJEICAO.md`, dentro da pasta de saída —
  ver [Artefato de saída](saida/index.md)) — ele já traz boa
  parte do diagnóstico.

Mande para quem te enviou este pacote, ou abra uma
[Issue](https://github.com/darthgrogu/herbflow-nir-collection-kit/issues)
no repositório no GitHub. Seu teste é o que torna a próxima versão melhor.
