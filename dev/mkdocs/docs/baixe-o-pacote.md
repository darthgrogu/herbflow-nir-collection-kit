# Baixe o pacote de teste

!!! tip "O pacote é baixado à parte do guia"
    Este guia (online ou dentro do próprio pacote) só documenta o fluxo. Para
    testar de verdade — planilha, gerador de artefato, datasets de exemplo
    — você precisa do **pacote completo** (`.zip`), baixado pela aba Releases.

## Baixar

O pacote de teste (`.zip`) contém tudo que você precisa: a planilha
(`.xlsm`), o gerador de artefato (`.exe`) e dois datasets de exemplo prontos
para testar sem coletar nada.

1. Baixe o `.zip` pelo botão abaixo (ou veja todas as versões na
   [aba Releases](https://github.com/darthgrogu/herbflow-nir-collection-kit/releases)).
2. Extraia o `.zip` inteiro numa pasta local.
3. Comece pelo [Primeiro teste](test-drive.md) — dá para ver o fluxo
   completo funcionando antes de coletar qualquer coisa de verdade.

[⬇ Baixar herbflow-nir-collection-kit-v0.1.0.zip](https://github.com/darthgrogu/herbflow-nir-collection-kit/releases/download/v0.1.0/herbflow-nir-collection-kit-v0.1.0.zip){ .md-button .md-button--primary }

## Atualizações

Este guia online é atualizado continuamente; o **pacote** (planilha +
gerador de artefato) segue sua própria cadência de versões, mais espaçada.
Para saber se há uma versão mais nova do pacote, confira a aba Releases, ou
veja as [Notas de versão](changelog.md).

!!! tip "Testando e quer dar retorno?"
    Erro, comportamento estranho, sugestão — qualquer coisa. Veja
    [Feedback](feedback.md) para saber o que reunir antes de reportar.
    **Qualquer feedback seu é valioso** para o aprimoramento da ferramenta —
    é exatamente o que este teste está procurando.
