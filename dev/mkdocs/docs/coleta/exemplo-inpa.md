# Coleta real: iHerbSpec + InnoSpectra + INPA

Esta é a demonstração concreta da [etapa de coleta](index.md): uma coleta real,
feita em parceria com o herbário do **INPA**, seguindo o protocolo
**iHerbSpec** e usando o equipamento **InnoSpectra (NIR-S-G1)**. É o exemplo
principal deste guia — e é justamente essa coleta que gera os dados que
alimentam a [conversão (parser)](../parser/index.md) no restante do fluxo. Os
[datasets de exemplo](../test-drive.md) do pacote vêm dela.
