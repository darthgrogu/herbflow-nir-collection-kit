# Requisitos e desbloqueio

## O que você precisa

- Um computador com **Windows** e **Microsoft Excel** instalados.
- Alguns minutos livres para o primeiro teste — pode ser com um dos datasets
  de exemplo incluídos no pacote, sem precisar de equipamento nenhum (ver
  [test-drive](../test-drive.md)).

## Antes de abrir a planilha: desbloqueie o arquivo

Como o `.xlsm` veio da internet (e-mail, link, zip), o Windows bloqueia as
macros por padrão — você vai ver um aviso, mas o botão "Habilitar Conteúdo"
**não resolve sozinho**. Faça isto primeiro:

1. Clique com o botão **direito** no arquivo `herbflow_iHerbSpec.xlsm` →
   **Propriedades**.
2. Na aba **Geral**, marque a caixa **Desbloquear** (perto do rodapé da
   janela) → **OK**.
3. Só então abra o arquivo e clique em **Habilitar Conteúdo** quando o Excel
   pedir.

Se você extraiu o `.zip` inteiro antes de fazer isso, o passo acima ainda
funciona — faça-o no arquivo já extraído, antes de abrir.
